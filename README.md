# MCP Learning Project

## 🎯 프로젝트 목표
Model Context Protocol (MCP)에 대한 이해와 실습을 위한 학습 저장소입니다.

## 📚 학습 내용
- MCP 기본 개념 및 구조
- MCP 서버/클라이언트 구현
- 다양한 도구(Tools) 개발

## 🚀 시작하기
```bash
# 저장소 클론
git clone https://github.com/NoirStar/mcp-learning.git
cd mcp-learning

# 의존성 설치
npm install  # 또는 pip install -r requirements.txt
```

## 📁 프로젝트 구조
```
mcp-learning/
├── docs/           # 학습 문서
├── examples/       # 예제 코드
├── src/           # 실습 코드
└── tests/         # 테스트 코드
```
